# Smooth Scroll Animation with Locomotive Scroll & GSAP

This project demonstrates a smooth scroll animation using **Locomotive Scroll** and **GSAP's ScrollTrigger**. The animation integrates scroll-based effects, image sequences, and pinning to create a visually engaging experience.

## 🚀 Features

- **Smooth Scrolling**: Powered by Locomotive Scroll for a seamless scroll experience.
- **Scroll-triggered Animations**: Uses GSAP's ScrollTrigger to animate content based on the scroll position.
- **Image Sequence Rendering**: Displays a sequence of images that change as the user scrolls, creating a fluid animation.
- **Responsive Design**: Adjusts to different screen sizes and ensures consistent animations.

## 📋 Prerequisites

Before you begin, make sure you have met the following requirements:

- You have installed **Node.js** (for setting up a local server).
- You have a basic understanding of **HTML**, **CSS**, and **JavaScript**.
- You have images named `male0001.png` to `male0300.png` in the project directory.

## 💻 Technologies Used

- **HTML5**: Structuring the content.
- **CSS3**: Styling and animations.
- **JavaScript**: Logic for animations and interactivity.
- **GSAP**: Powerful JavaScript library for animations.
- **Locomotive Scroll**: For smooth scrolling and scroll-based effects.

🛠 Setup & Installation

1. Clone the repository:
git clone https://github.com/anshulkotwal/3d-Websites-projects/new/main/korean_boy_effect


2. Install Locomotive Scroll & GSAP:

You can use a CDN or install via npm.

Using CDN: Add the following lines to your HTML file:
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.0/gsap.min.js"></script>
<script src="https://unpkg.com/locomotive-scroll/dist/locomotive-scroll.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.0/ScrollTrigger.min.js"></script>


Using npm:
npm install gsap locomotive-scroll


3. Run the project:

You can use Live Server in VS Code or any other local server to open index.html.
