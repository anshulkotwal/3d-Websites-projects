# 🌌 3D Interactive Solar System Simulation

A visually stunning and interactive 3D solar system simulation built using **Three.js** and **GSAP**. This project brings to life the beauty of space with rotating planets, smooth animations, and a dynamic environment.

## 🎨 Features
- **Realistic Space Environment**: Uses HDR textures for immersive lighting and reflections.
- **Interactive Animations**: Smooth scrolling animations to rotate and display planets.
- **Orbiting Planets**: Planets are positioned in a circular orbit around a central point.
- **Responsive Design**: Adapts to window resizing, ensuring a consistent experience across devices.

## 🛠️ Technologies Used
- **Three.js**: For creating and rendering the 3D environment.
- **GSAP**: For smooth and interactive animations.
- **RGBELoader**: For loading high dynamic range (HDR) environment textures.

## 🚀 Getting Started
### Prerequisites
Make sure you have **Node.js** installed on your system.

### Installation
1. Clone the repository:
    ```bash
    https://github.com/anshulkotwal/3d-Websites-projects/tree/main/The%20Planets
    ```
2. Navigate to the project directory:
    ```bash
    cd 3d-Websites-projects
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

### Usage
1. Start the development server:
    ```bash
    npm start
    ```
2. Open `http://localhost:3000` in your browser to view the simulation.
